# README
### Реализовать функцию поиска
* по вопросам, 
* ответам, 
* комментариям,
* пользователям. 
* Должна быть возможность совершить общий поиск, сразу по всем возможным объектам,
* а также возможность поиска отдельно по вопросам/ответам/комментариям/пользователям.
